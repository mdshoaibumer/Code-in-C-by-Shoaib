# 🛠️ Project Name: Task CLI in C

## 📌 Description
A command-line task manager built in pure C using file-based persistence. Supports add/remove/list tasks with priorities.

## 🚀 Features
- Add and list tasks
- Save/load from file
- Sort by priority

## 🧠 Concepts Used
- File I/O
- Structs and dynamic memory
- Command-line parsing
